What it does

One creator, one channel, one publish button

Spike Publisher is not a platform and has no users. It is the private release step of a single content pipeline — it moves finished files from the creator's machine to the creator's own accounts, correctly labeled, at the right time.

1

Staged uploads

Each video is prepared and staged with its title, description, captions and thumbnail, then reviewed before anything leaves the machine.

▶

One-button GO

A single confirmed action pushes the staged upload to YouTube, TikTok or Instagram through their official Content Posting APIs.

⏱

Engine-side scheduling

Publish times are set at upload using each platform's native scheduling, so episodes go live at the planned slot without manual re-visits.

◎

AI-content labeling

The channel is animated and partly AI-assisted, so every upload carries the platform's AI-generated / altered-content disclosure by default.

≡

Captions attached

Karaoke-style captions authored in the pipeline travel with the video, so accessibility isn't a manual afterthought.

✓

Owner accounts only

The tool authenticates against the creator's own accounts and touches nothing else — no third-party users, no audience data.

How it works

From finished render to live episode

The publishing engine is a command-line tool run by the operator. It follows the same four steps every time.

1

Prepare

A finished video plus its metadata manifest (title, description, captions, disclosure flags, target platforms and publish time) is assembled from the production pipeline.

2

Stage

The engine uploads the file to the platform in a private / unpublished state and validates that titles, captions and AI labels were accepted before proceeding.

3

Schedule & GO

The operator confirms, and a single action sets the native publish time (or publishes now). Nothing is posted without that explicit confirmation.

4

Verify

The engine reads back the platform's API response to confirm the video exists and is scheduled correctly, then records the result.

Finished render→Stage (private)→Confirm GO→Scheduled & live
